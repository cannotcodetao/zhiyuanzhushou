**说明**：上述 XLS 为浙江省教育考试院官方发布的"普通类第一段/第二段平行投档分数线表"，是浙江高考投档录取的最终权威数据。本系统主要采用**第一段（平行录取一段）**数据，共 3 个年份（2023/2024/2025），覆盖 83 所在浙招生高校、2358 个专业、5927 条分数记录。

### 2. 辅助数据源：掌上高考学校元数据

| 内容 | URL | 用途 |
|------|-----|------|
| 全国高校基本信息 | https://static-data.gaokao.cn/www/2.0/school/list_v2.json | 学校层级(985/211/双一流)、所在城市、办学性质 |

**说明**：用于补充省外重点高校的层级、城市、标签等元数据。浙江省内高校的城市/层级以本项目"浙江高校参照表"（基于教育部高校名单 + 浙江省教育厅公开信息整理）为准。

### 3. 字段推导规则（derived fields，非编造）

`subject_req`（选科要求）与 `category`（学科门类）两个字段在官方投档分数线 XLS 中不存在，依据以下**官方分类标准**按专业名称规则推导：

| 字段 | 推导依据 | 文件 |
|------|----------|------|
| `subject_req` | 教育部《普通高校本科招生专业选考科目要求指引》+ 浙江省 2024-2025 年选科实际（理工医农普遍要求物化同选） | `raw/build_data.py` 中 `derive_subject_req()` |
| `category` | 教育部《普通高等学校本科专业目录（2024 年）》学科门类划分 | `raw/build_data.py` 中 `derive_category()` |

## 二、字段映射关系

### 官方 XLS → JSON 字段映射

| XLS 列名 | JSON 字段 | 说明 |
|----------|-----------|------|
| 学校代号 | — | 用于学校识别，未直接存储（学校 ID 由本系统重新分配） |
| 学校名称 | `school.name` | 直接对应 |
| 专业代号 | — | 不存储 |
| 专业名称 | `major.name` | 直接对应（跨年同名专业取最详细版本） |
| 计划数 | `score.enrollment` | 该专业当年执行计划数 |
| 分数线 | `score.min_score` | 该专业当年最低投档分 |
| 位次 | `score.min_rank` | 该专业当年最低录取位次 |

### list_v2.json → JSON 字段映射

| list_v2 字段 | JSON 字段 | 说明 |
|--------------|-----------|------|
| `name` | `school.name` | 学校名称匹配键 |
| `f985` (1/2) | `school.level` / `tags` | f985=1 → level="985" |
| `f211` (1/2) | `school.level` / `tags` | f211=1 → level="211" |
| `dual_class` (1/2) | `tags` | =1 → 含"双一流"标签 |
| `c` (城市) | `school.city` | 去"市"后缀（如"杭州市"→"杭州"） |
| `p` (省份) | `school.province` | 学校所在省份 |
| `nature` | `tags` | "民办" → 含"民办"标签 |

### 选科要求推导规则要点

- 工学/理学/医学(临床口腔药学)/农学 大类 → `["物理","化学"]`（2024 教育部改革后浙江实际）
- 工科试验班/理科试验班 → `["物理","化学"]`
- 护理学 → `["生物"]`
- 地理科学类 → `["地理"]`
- 思政/马克思主义 → `["政治"]`
- 历史学/考古 → `["历史"]`
- 文/经/管/法/教/艺术 大类 → `[]`（不限，与前端 `checkSubjectMatch` 语义一致：空数组表示所有考生均可报考）
- 试验班含人文社科方向 → `[]`

## 五、已知数据缺失与质量问题

1. **选科要求为推导字段**：`subject_req` 依据专业名称规则推导，非逐校逐专业从官方招生计划核对。个别学校的特殊要求（如某校建筑学要求物理、另一校不限）可能存在偏差。**建议考生最终以浙江省教育考试院"浙江省高校招生考试信息管理系统"发布的招生计划为准。**

2. **学科门类为推导字段**：`category` 依据教育部专业目录按专业名归类，对于"试验班""大类招生"等交叉专业，归类可能存在主观判断（如"社会科学试验班"归为"法学"门类）。

3. **部分专业某年数据缺失**：若某专业在某年未在浙江一段招生（或招生计划为 0），该年份记录省略。这是正常现象，不代表数据错误。

4. **专业名跨年不完全一致**：同一专业在不同年份的官方 XLS 中名称可能略有差异（如 2024 "新闻传播学类" vs 2025 "新闻传播学类（含智能与国际传播创新班）"）。本系统通过去除括号内容做归一化匹配，取最详细名称作为展示名。

5. **二段数据未纳入主数据集**：本系统目前仅采用"平行录取一段"数据（覆盖重点本科及普通本科主要招生）。二段 XLS 已下载备用（`raw/zhejiang_2025_phase2.xls`），如需扩展可补充。

6. **北京大学 2024 年 721 分**：略高于验证脚本设定的 720 上限阈值，经核对官方 XLS 确认为真实数据（顶尖高校在浙投档分可超 720），非数据错误。

7. **西湖大学、浙江警察学院等未纳入**：西湖大学 2025 年可能在提前批招生；浙江警察学院属公安类提前录取；二者均不在"平行录取一段"XLS 中，故未纳入数据集。

## 六、可追溯性验证

每条分数记录均可通过 **(学校名称, 年份, 专业名称)** 三元组回溯到对应年份的官方 XLS 原始行。验证脚本 `validate_data.py` 的抽样核对功能已确认 10 条随机样本与官方 XLS 100% 一致（分数、位次均完全匹配）。

1. 本系统的分数、位次、计划数数据 100% 来自浙江省教育考试院官方发布的投档分数线表，数据真实可追溯。
2. 选科要求（`subject_req`）与学科门类（`category`）为依据教育部公开标准按专业名推导的字段，仅供参考，**不构成官方招生承诺**。
3. 本系统数据**仅供志愿填报参考**，最终录取数据以各省教育考试院公告及高校正式录取通知为准。
4. 考生在正式填报志愿前，务必查阅浙江省教育考试院官方系统（www.zjzs.net）发布的最新招生计划与选科要求，并仔细阅读拟报考高校的招生章程。
5. 数据获取时间为 2026-06-30，如官方后续发布更新或勘误，本系统数据可能未同步。
