在推荐结果的冲/稳/保卡片上，点击「展开专业」按钮即可查看该大学所有位次达标 + 选科匹配的专业列表。每个专业显示近 3 年录取分数 / 位次 / 选科要求 / 可报年份。

功能详解 · 2/5 单科优势

基于得分率横向比较各科

1
得分率法

各科分数 ÷ 满分，实现不同满分科目横向比较（语数英 150 / 选考 100）。

2
优势识别

高于平均分 +5% 为优势学科，低于 -5% 为薄弱学科。

3
针对性推荐

10 门学科各对应 4 个优势专业方向 + 风险专业提示。

让强项真正发挥价值

输入各科成绩，系统自动识别优势学科和薄弱学科。例如物理得分率 85% 被识别为优势 → 推荐工科/物理类方向；历史得分率 62% 被识别为薄弱 → 提示避免历史学等相关专业。少于 2 科不触发分析（优雅降级），各科均衡时不强加推荐。

功能详解 · 3/5 三方向建议

政策 / 就业 / 兴趣三方向专业建议

教育部权威

政策导向

基于教育部《本科专业目录（2025/2026 年）》、国家战略性新兴产业 9 大方向、工信部直属高校 2026 年新增 74 个专业（低空经济 / 具身智能 / 量子信息 / 集成电路等）。根据用户选科自动匹配可报专业。

麦可思报告

就业导向

基于麦可思《2025 年中国本科生就业报告》。绿牌 6 个（电气 / 微电子 / 机械电子 / 新能源 / 车辆 / 机器人），红牌 5 个（公共事业管理 / 音乐表演 / 绘画 / 法学 / 美术学），高薪 Top10（微电子 7814 元 / 电子科学 7752 / 自动化 7573 等）。

霍兰德理论

兴趣导向

基于选科组合推导兴趣倾向（霍兰德职业兴趣理论 + 教育部学科认知）。物化生 → 医学 / 科研，物化地 → 工科，物化技 → 信息技术等 10 条选科组合兴趣方向。用户也可主动选择 10 个兴趣方向芯片。

三方向建议互补而非互斥。政策导向看未来趋势，就业导向看现实回报，兴趣导向看个人适配。系统综合三者给出立体化的专业推荐。
功能详解 · 4/5 兴趣导向

让兴趣成为选专业的第一驱动

1
主动选择优先

用户可从 10 个兴趣方向芯片中选择（IT / 医学 / 金融 / 教育 / 设计 / 机械 / 法律 / 经管 / 材料 / 传媒）。

2
选科推导 fallback

未选择时，基于选科组合智能推导（霍兰德职业兴趣理论）。

3
优先级推荐

系统优先推荐匹配用户兴趣的专业，未匹配专业不被过滤但靠后展示。

兴趣是坚持的动力

10 个兴趣方向覆盖主要职业领域。用户主动选择时系统优先推荐匹配专业；未选择时 fallback 到基于选科组合的智能推导。例如选科物化生 → 推导兴趣方向为医学 / 科研 / 工程。

功能详解 · 5/5 志愿表

一键管理你的志愿清单

快速操作

一键加入

在推荐结果的每个专业卡片上，点击「加入志愿表」按钮即可加入。已加入的显示「已加入志愿表」状态，可重复点击切换。

集中管理

志愿表查看

所有加入的专业集中展示在志愿表面板，按加入顺序排列。每个志愿显示学校名 / 专业名 / 录取概率 / 推荐档位（冲 / 稳 / 保）。

PDF 导出

打印导出

浏览器原生打印功能，支持导出 PDF。打印样式已优化，去除交互元素，保留核心信息（学校 / 专业 / 概率 / 理由），适合家长会商和线下参考。

志愿表数据仅保存在本地浏览器，不上传任何服务器，保护考生隐私。关闭浏览器后志愿表清空，建议打印或导出 PDF 保存。
推荐算法 位次法

基于位次的科学推荐

1
收集候选

筛选选科匹配 + 近 3 年有录取数据的专业。

2
加权位次

按年份权重计算平均位次（最近年份权重高）。

3
概率计算

ratio = 专业位次 / 学生位次，越小越难考。

4
冲稳保分配

滑动窗口找最接近学生位次的 5 所做稳妥，上下各取 3 所做冲保。

ratio < 0.9　冲刺
0.9 – 1.2　稳妥
≥ 1.2　保底

位次法是各省考试院官方投档依据，比分数法更精确（同分不同年份位次差很大）。
