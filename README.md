# 志愿智选 — 高考志愿智能规划系统

> 基于全国 31 省官方录取数据，用位次法为你科学推荐冲稳保院校专业。

**志愿智选**是一个开源的纯前端高考志愿规划工具。输入分数和位次，系统基于各省考试院官方投档数据，通过位次法计算录取概率，自动分配冲刺 / 稳妥 / 保底三档推荐。覆盖全国 31 个高考省份，支持新高考 3+3、3+1+2 和老高考文理分科三种模式。

## 六大核心功能

### 1. 智能推荐（冲稳保）
输入分数、位次和选科，系统通过位次法计算每个专业的录取概率，自动分配 **冲刺 3 所 / 稳妥 5 所 / 保底 3 所** 三档推荐。每所院校附录取概率、近 3 年录取数据和推荐理由。

### 2. 展开专业（历史查询）
在推荐结果的每所大学卡片上，点击「展开专业」即可查看该大学所有 **位次匹配 + 选科匹配** 的专业列表。近 3 年数据合并去重，任一年能上即显示，并标注可报年份。

### 3. 专业建议（三方向）
- **政策导向**：教育部《本科专业目录（2025/2026 年）》、国家战略性新兴产业 9 大方向、工信部直属高校 2026 年新增专业
- **就业导向**：麦可思《2025 年中国本科生就业报告》绿牌 / 红牌 / 高薪 Top10 专业
- **兴趣导向**：10 个兴趣方向芯片（IT / 医学 / 金融 / 教育 / 设计 / 机械 / 法律 / 经管 / 材料 / 传媒），基于选科组合智能推导

### 4. 单科优势分析
输入各科成绩，系统基于得分率（分数 ÷ 满分）横向比较不同满分科目，识别优势学科和薄弱学科，给出针对性专业建议。支持 10 门学科各对应 4 个优势专业方向 + 风险专业提示。

### 5. 兴趣导向推荐
用户可主动选择兴趣方向，系统优先推荐匹配专业。未选择时 fallback 到基于选科组合的霍兰德职业兴趣理论推导。

### 6. 志愿表管理
一键将推荐专业加入志愿表，支持打印导出 PDF（浏览器原生打印功能）。

## 技术架构

### 配置驱动设计
通过 `js/province-config.js` 集中定义 31 省配置（考试模式、科目、总分、数据文件路径等），算法层零修改。扩展新省份只需：
1. 添加省份配置
2. 放入数据 JSON 文件

### 三种高考模式

| 模式 | electiveMode | 覆盖省份 | 选科规则 | 总分 |
|------|-------------|----------|----------|------|
| 新高考 3+3 | `free` | 浙江 / 山东 / 上海 / 北京 / 天津 / 海南 | checkbox 多选 3 门 | 750（上海 660 / 海南 900） |
| 新高考 3+1+2 | `layered` | 江苏 + 第三批 8 省 + 第四批 7 省 + 第五批 8 省 | radio 首选物理 / 历史 + checkbox 再选 2 门 | 750 |
| 老高考文理分科 | `old` | 新疆 / 西藏 | radio 选文科 / 理科 | 750 |

### 推荐算法
- **位次法**：`ratio = 专业位次 / 学生位次`，ratio 越小专业越难考
- **冲稳保阈值**：ratio < 0.9 冲刺 / 0.9-1.2 稳妥 / ≥1.2 保底
- **加权平均位次**：按年份权重计算（最近年份权重高）
- **学校去重**：每所学校只保留与学生位次最接近的一个专业
- **滑动窗口分配**：找平均位次差最小的 5 所做稳妥，上下各取 3 所做冲保

## 数据规模

| 指标 | 数量 |
|------|------|
| 覆盖省份 | 31 省 |
| 高校数量 | ~1100 所 |
| 专业数量 | ~38000 个 |
| 录取记录 | ~96000 条 |
| 数据年份 | 2023-2025（第五批新高考省份为 2025 单年） |

### 数据来源（100% 官方可追溯）

每条录取记录可回溯到省考试院原始 XLS 文件。详细数据来源见 [`data/DATA_SOURCES.md`](data/DATA_SOURCES.md)。

- **浙江省**：浙江省教育考试院 [www.zjzs.net](https://www.zjzs.net)
- **山东省**：山东省教育招生考试院 [www.sdzk.cn](https://www.sdzk.cn)
- **江苏省**：江苏省教育考试院 [www.jseea.cn](https://www.jseea.cn)
- **湖南省**：湖南省教育考试院 [hneeb.cn](https://hneeb.cn)（通过一分一段表反查补全位次）
- **其余 27 省**：各省考试院官方投档线 XLS（详见 DATA_SOURCES.md）

## 本地使用

### 方式一：直接打开
1. 克隆仓库：`git clone https://github.com/cannotcodetao/zhiyuanzhushou.git`
2. 用浏览器打开 `index.html`

### 方式二：本地服务器（推荐）
```bash
cd zhiyuan-zhixuan
python -m http.server 8080
# 浏览器访问 http://localhost:8080/
```

### 部署到 GitHub Pages
1. Fork 本仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main` / `/ (root)`
5. 点击 Save，等待 1-3 分钟即可上线

## 设计风格

采用 **Blue Professional** 设计系统 — 咨询级专业感，适合教育数据产品：
- **色彩**：奶油画布 `#fdfae7` + 钴蓝主色 `#1e2bfa`（单一强调色，克制专业）
- **字体**：思源黑体（标题 / 数字）+ 思源宋体（正文）
- **卡片**：4% 钴蓝填充 + 20% 钴蓝边框 + 14px 圆角，无投影
- **交互**：胶囊形按钮、柔和悬浮、入场动画
