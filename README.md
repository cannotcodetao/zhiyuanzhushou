# 志愿智选 — 想法验证与MVP规划总览

## 项目概览

- **项目名称**：志愿智选 — 高考志愿智能规划系统
- **一句话描述**：输入高考分数和位次，输出冲稳保学校推荐清单
- **核心用户**：分数不高不低、怕浪费分数又怕滑档的高考考生家长
- **当前阶段**：想法验证期（手动服务验证）

---

## 验证结论

**🟡 需要更多验证（但前景很好）**

### 绿旗（好信号）✅
1. 人们已经在为更差的解决方案付费（志愿卡200-500元）
2. 社区里充满抱怨（每年6月全网焦虑）
3. 一句话能说清用户和痛点
4. 时间窗口集中，获客效率高
5. 可以手动开始，零代码验证

### 红旗（风险）⚠️
1. 季节性太强 → 拓展考研/留学等相邻赛道
2. 数据准确性责任 → 强烈免责声明 + 仅供参考
3. 信任建立难 → 真实案例 + 透明逻辑
4. 巨头竞争 → 差异化定位
5. 政策变化风险 → 每年更新数据

---

## 三阶段路线图

```
Stage 1: 手动服务 → Stage 2: 流程化 → Stage 3: 产品化
   (99元/次)        (SOP+兼职)       (自助查询9.9元)
```

### Stage 1: 手动服务（立即开始）
- **做什么**：1对1志愿填报咨询
- **工具**：Excel + 腾讯文档 + 微信
- **定价**：99元/次
- **目标**：服务10个付费用户，验证需求

### Stage 2: 流程化（服务10人后）
- **做什么**：整理SOP，雇兼职执行
- **核心产出**：《志愿智选咨询SOP v1.0》
- **定价**：199-299元/次
- **目标**：月服务50单，质量稳定

### Stage 3: 产品化（验证通过后）
- **做什么**：纯前端HTML+JS的自助查询工具
- **核心功能**：输入分数→输出冲稳保清单
- **定价**：9.9元/次 查询，49元/份 报告
- **目标**：首月100+付费用户
